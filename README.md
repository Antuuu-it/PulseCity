PulseCity

Descripción del Proyecto
PulseCity es una aplicación de escritorio desarrollada en C# utilizando Windows Forms, cuyo objetivo es permitir el registro y visualización de reportes ciudadanos relacionados con diferentes problemáticas urbanas.
El sistema permite registrar incidencias de tráfico, contaminación, basura, agua, seguridad y ruido, indicando la zona afectada y el nivel de gravedad correspondiente.

Objetivos
Objetivo General
Desarrollar una aplicación de escritorio que permita registrar y visualizar reportes ciudadanos mediante una interfaz gráfica sencilla e intuitiva.
Objetivos Específicos
* Implementar una interfaz gráfica utilizando Windows Forms.
* Permitir registrar reportes indicando tipo, zona y nivel.
* Almacenar los reportes mediante listas.
* Visualizar la información mediante un DataGridView.
* Aplicar conceptos de Programación Orientada a Objetos.

Tecnologías Utilizadas
* Lenguaje: C#
* Framework: .NET
* Entorno de desarrollo: Visual Studio
* Interfaz gráfica: Windows Forms
* Control de versiones: Git y GitHub

Instrucciones de Instalación
1. Clonar el repositorio:
2. Abrir el archivo `PulseCity.sln` en Visual Studio.
3. Restaurar las dependencias del proyecto si es necesario.
4. Compilar la solución.

Instrucciones de Ejecución
1. Abrir el proyecto en Visual Studio.
2. Seleccionar PulseCity como proyecto de inicio.
3. Presionar F5 o el botón Iniciar
4. Utilizar el Dashboard para acceder a las funcionalidades del sistema.
